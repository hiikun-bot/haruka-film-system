プライバシーポリシー

HARUKA FILM SYSTEM

1. はじめに

HARUKA FILM SYSTEM（以下「本サービス」）は、HARUKA FILM（以下「当社」）が運営する社内向け業務管理ツールです。本ポリシーは、本サービスにおけるユーザー情報の取得・利用・管理について定めるものです。

2. 取得する情報

本サービスは、利用目的の達成に必要な範囲で以下の情報を取得します。

  • アカウント情報: 氏名、ニックネーム、メールアドレス、所属チーム、役割（ロール）
  • 業務情報: 案件・クリエイティブ・請求・チーム稼働時間に関する業務データ
  • Google Calendar 連携情報（任意）: ユーザーが明示的に連携を許可した場合に限り、Google Calendar の予定の時間情報のみ（開始日時・終了日時・全日扱いか否か）を取得します。予定のタイトル、説明、場所、参加者、招待元、添付ファイル、ビデオ会議リンク等の「内容」は取得しません。これは API リクエスト段階で取得フィールドを時間情報のみに制限する実装によって担保されています。
  • 利用ログ: アクセス日時、IPアドレス、ブラウザ情報等

3. 利用目的

  • 業務管理（案件・クリエイティブ・請求・稼働時間）
  • チームメンバー間の稼働可能時間の可視化と工数調整
  • サービス改善のためのログ分析
  • 不正利用の検知・防止

4. Google API Services User Data Policy への準拠

本サービスが Google API から取得する情報は、Google API Services User Data Policy の Limited Use 要件を含め、同ポリシーに準拠して取り扱います。

HARUKA FILM SYSTEM's use and transfer to any other app of information received from Google APIs will adhere to the Google API Services User Data Policy, including the Limited Use requirements.

具体的には:

  • Google Calendar から取得した時間情報は、本サービスのチーム稼働時間表示・工数調整目的にのみ利用します
  • 当該情報を広告目的で利用しません
  • 当該情報を人間が閲覧する形で他者に提供しません（本サービス内のチーム稼働時間グリッド表示等、ユーザーが期待する機能の範囲を除く）
  • 当該情報を第三者に販売・譲渡しません
  • 当該情報を機械学習モデルの学習には利用しません

5. 第三者提供

取得した情報は、以下の場合を除き、ユーザー本人の同意なく第三者に提供しません。

  • 法令に基づく開示要請がある場合
  • 人の生命・身体・財産の保護のために必要であり、本人の同意取得が困難な場合
  • サービス提供に必要な業務委託先（クラウドインフラ事業者等）に対し、必要最小限の範囲で提供する場合

6. 利用する外部サービス（業務委託先）

  • Supabase: データベース・認証基盤
  • Railway: アプリケーション実行環境
  • Google Cloud (OAuth / Calendar API / Drive API): 認可および各種連携
  • Slack: 業務通知

7. 保管と保護

  • Google OAuth の refresh_token は AES-256-GCM で暗号化して保管します
  • 取得した予定の時間情報は当社管理下のデータベースに保管し、アクセスは認証済みの本サービス利用者に限定します
  • 通信は全て HTTPS で暗号化されます
  • アクセス権限はロールに基づき必要最小限に制限されます

8. ユーザーの権利

  • いつでも本サービス内の「連携を切断」ボタンから Google Calendar 連携を解除できます。解除と同時に refresh_token はデータベースから削除されます
  • 蓄積された個人データの開示・訂正・削除を希望する場合は、下記の問い合わせ先までご連絡ください
  • Google アカウント側からも https://myaccount.google.com/permissions で本サービスのアクセス権を取り消すことができます

9. データ保存期間

  • アカウント情報: 利用継続中。退会後は削除または匿名化します
  • 業務情報: 業務管理上必要な期間（法令で求められる場合はその期間）
  • Google Calendar 連携情報: 連携解除まで（解除後即時削除）
  • 利用ログ: 最大1年

10. Cookie 等の利用

本サービスはセッション管理のために Cookie を使用します。これらは認証状態の維持に必要な技術的 Cookie であり、広告目的・トラッキング目的の Cookie は使用しません。
