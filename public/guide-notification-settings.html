💬 チャット通知（Slack / Chatwork）

Dチェック依頼・修正依頼・納品完了などの「仕事のバトンパス」が、案件の Slack チャンネル / Chatwork ルームに、あなた宛のメンション付き で投稿されます。普段使っているチャットに届くので、システムを開いていなくても気づけます。
👉 届くようにするには STEP 2 の DM ID 登録 が必須。

🔔 ベル通知（アプリ内通知）

システムのヘッダー右上にある ベルアイコン 🔔 に届く通知です。未読件数バッジが付き、開くと一覧がスライドインします。ログインしていれば設定なしで届く のが特徴（詳しくは STEP 4）。

なぜ両方あるの？

ベル通知は「システムを開いている人」向け、チャット通知は「システムを開いていない人」向け。DM ID を入れておけば二重の網 で連絡漏れを防げます。

1💬 Slack DM ID
なぜ入れる？

システムが あなた宛て に Slack メンションを付けるための宛先 ID。

入れるとどう変わる？

Dチェック依頼／修正通知／納品完了が あなた宛メンション付きで Slack に届く。「気づかずスルー」がなくなる。

📋 例：あなたのクリエイティブが「Dチェック」に進んだ瞬間、ディレクターに Slack 通知 → 確認スピードアップ
取得方法

Slack の DM 画面 → プロフィール右クリック → 「メンバーIDをコピー」。U から始まる 10 桁前後の文字列。

2💬 Chatwork DM ID
なぜ入れる？

Chatwork メンションの あなたの番号。Chatwork を主に使うクライアント・チームで重要。

入れるとどう変わる？

Chatwork の自分宛てに通知が飛ぶ。Slack 派／Chatwork 派どちらでも漏れなく届く。

📋 例：Chatwork メンション形式 [To:6603336] の 数字部分 を入力
取得方法

Chatwork のメンションをコピペ、または プロフィール画面の「アカウント ID」を確認。

ここだけは絶対！

DM ID が空欄の人への通知は スキップされます（メンションできない人に投げても本人が気づけないため）。「通知が来ない」の原因 No.1 はここ。Slack・Chatwork 両方 入れておくのがおすすめです。

✅
ID 登録、完了！これであなた宛のバトンパスが届くようになります

STEP 3

いつ・誰に届く？（チャット通知の早見表）

⏱ 約 1 分
クリエイティブのステータスが変わった瞬間に飛ぶ

クリエイティブの ステータス遷移 がトリガーです。「次に動く人」へ自動でバトンが渡ります。

ステータスがこうなったら誰に届く？
Wチェック📥Wチェック担当者 に「Wチェック依頼」
Dチェック📥ディレクター に「Dチェック依頼」
Pチェック📥プロデューサー に「Pチェック依頼」
Wチェック後修正 / Dチェック後修正 / Pチェック後修正 / クライアントチェック後修正🔁編集者・デザイナー（制作担当） に「修正依頼」
クライアントチェック中🎉制作担当 に「クライアントチェックに進みました」
納品🎉編集者・デザイナー に「クライアントOK！納品完了」

💡 通知にはクリエイティブ詳細への URL が入っているので、クリックすればそのまま該当画面が開きます。

どこに投稿される？

Slack は 案件に設定された Slack チャンネル（未設定ならクライアントのチャンネル）へ、あなたの <@メンション> 付きで投稿。
Chatwork は 案件の Chatwork ルーム（未設定ならクライアントのルーム）へ、[To:あなたのID] 付きで投稿されます。

案件側の設定も必要

あなたの ID だけでなく、案件（またはクライアント）に Slack チャンネル URL / Chatwork ルーム ID が設定されている ことが前提です。案件側が未設定だとチャット通知は飛びません。設定はディレクター・秘書チームに確認を。

ベル通知 🔔 の使い方

⏱ 約 1 分
ヘッダー右上のベルに、こんな通知が届く
通知の種類いつ届く？
⚪ボールが返ってきましたクリエイティブのボール（次に動く番）があなたに移った瞬間
📢全体通知管理者・秘書・プロデューサーからの全体連絡
@メンションつぶやきやコメントで @あなた が呼ばれたとき
❤️リアクションあなたのつぶやきにリアクションが付いたとき
💬コメントあなたの投稿・コメントに返信が付いたとき
🎬クリエイティブ関連ステータス遷移の連絡や、新規登録の連絡（新規登録は管理者・秘書向け）

未読があるとベルに 件数バッジ（100件以上は「99+」）が付き、新着は 画面を開いたままリアルタイム で飛び込んできます。パネルの「✓ すべて既読」でまとめて既読にできます。

📢 全体通知は「稼働時間」に合わせて届くことがある

送信者が時間帯指定モードを使うと、全体通知は あなたのプロフィールの稼働時間の開始タイミング に合わせて配信されます。深夜に通知が鳴らないのはこの仕組みのおかげ。稼働時間の設定は はじめての登録チュートリアル STEP 2 を参照。
