TUTORIAL ／ はじめての方へ

ようこそ、HARUKA FILM へ！このチュートリアルを進めれば、すぐに使い始められます

3 つのステップを順番にやるだけ。途中で抜けても大丈夫、続きから戻れます。

⏱ 所要時間：約 5 〜 10 分
準備OK ／ 0 / 3
1
STEP 1

アカウントを作る

⏱ 約 2 分
届いた招待リンクを開く

事務局（秘書）から Chatwork や LINE などで https://...invite.html?token=... のリンクが届いているはずです。タップ／クリックして開いてください。

⏰ 24時間ルール： リンクは発行から 24時間で無効 になります。期限切れだったら「リンクを再発行してください」と事務局に伝えてください。
表示内容を確認する

登録画面の上部に「📩 招待が届いています」の下に あなたのメールアドレス と 付与されるロール が出ます。間違いがないか必ずチェック。

❌ 違っていたら登録しないで！ その場で事務局に連絡してください。誤った状態で登録してしまうと、あとから直してもらう手間がかかります。
表示名・パスワードを入力

3 項目を入れます。パスワードは 8 文字以上。あとで変えられるので、今は仮のもので OK。

入力イメージ
表示名 * パスワード * パスワード（確認）
💡 表示名のコツ： 他のメンバーから見える名前です。普段呼ばれている名前（田中、たろ、はなちゃんなど）にすると、メンションされやすくなります。
✅
アカウント作成、おつかれさま！自動でログインされて、ダッシュボードが開きます
STEP 2 へ進む
2
STEP 2

プロフィールを整える

⏱ 約 3 分
プロフィール画面を開く

画面右上の アバター（イニシャル丸） をクリックすると、自分のプロフィール編集モーダルが開きます。スマホはハンバーガーメニュー →「👤 メンバー」から自分の行を編集。

「基本情報」タブを埋める
項目
説明
📷 アバター
「アイコンを変更」から画像をアップロード。一覧画面で識別しやすくなります。
推奨
ニックネーム
検索・フィルタで使える別名（例：たろ、はなちゃん）。普段呼ばれている名前を入れると探しやすい。
任意
職種
動画 ／ デザイン ／ 両方 から選択。クリエイティブのアサイン候補に影響します。
推奨
誕生日
月日のみ表示にすることも可（年齢非公開）。社内の誕生日通知に使われます。
任意
「稼働時間」タブで稼働可能な時間帯を入力

平日／土日祝 それぞれの稼働可能時間帯を入れます。アサイン時の稼働判定に使われます。「土日祝は稼働なし」のチェックも可。

💡 メモ欄のすすめ： スキル（After Effects / Premiere / Photoshop など）や得意ジャンル、連絡可能時間を書いておくと、アサインが回ってきやすくなります。
⭐ 「連絡先」タブ ── ここがいちばん大事

あなたに 通知（DM）が届くため に必要な ID を入力します。これを入れないと、Dチェック依頼や修正通知が あなたに飛びません。一覧画面で気づくしかなくなるので、必ず登録を。

項目
取得方法
Slack DM ID
Slack の DM 画面でプロフィール右クリック →「メンバーIDをコピー」
例：U0123456789
推奨
Chatwork DM ID
Chatwork のメンション形式 [To:6603336] の数字部分。
プロフィール画面でも確認できます。
推奨
✅
プロフィール初期設定、完了！これで他のメンバーからもあなたが見えるようになり、通知も届きます
STEP 3 へ進む
3
STEP 3

口座情報を登録する

⏱ 約 3 分
なぜ口座情報を入れるの？

あなたが納品したクリエイティブは、自動的に 請求書ドラフト になり、月末にあなた自身が「提出」ボタンを押すと管理者の承認待ちに入ります。
承認後の振込先として、口座情報が必要です。編集者・デザイナーの方は必須です。

「口座情報」タブを埋める
項目
例
銀行名 / コード
例：楽天銀行 / 0036
必須
支店名 / コード
例：ワルツ支店 / 204
必須
口座種別 / 番号
普通 / 当座、7桁の口座番号
必須
口座名義（カナ）
全角カナ、姓と名の間に半角スペース。例：タカハシ サトル
必須
電話番号 / 住所
請求書に印字されます。
推奨
⚠️ 名義カナは厳密に： 銀行口座の名義（カナ）と1文字でも違うと振込エラーになります。通帳・キャッシュカードを見ながら正確に入力してください。
✅
口座情報、登録完了！これで請求書がスムーズに処理されます

💬 困ったときは

招待リンクが期限切れになっていた
招待リンクは 24 時間で無効になります。事務局（秘書）に 「リンクを再発行してください」 と伝えてください。
「すでに使用されています」と表示された
そのリンクはすでに登録に使われています。ログイン画面からメール ＋ パスワードで入ってみてください。パスワードを忘れた場合は、事務局にパスワードリセットを依頼してください。
登録したのに通知（DM）が来ない
プロフィールの 「連絡先」タブ で Slack DM ID または Chatwork DM ID が空欄になっていないか確認してください（STEP 2 参照）。それでも来ない場合は事務局に相談を。
ロールが間違っていた
登録自体はそのままで OK。事務局に依頼すれば、メンバー編集画面でロールを変更してもらえます。アカウントを作り直す必要はありません。
それ以外で困ったとき
事務局（秘書）または管理者に Chatwork / LINE で気軽に聞いてください。「○○ができない」と画面のスクショを添えると早いです。
