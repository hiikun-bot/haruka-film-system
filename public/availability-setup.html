セキュリティメモ

  • refresh_token は AES-256-GCM で暗号化して DB 保管。
  • OAuth スコープは calendar.events.readonly + userinfo.email のみ。書き込み不要。
  • 取得するのは時間情報のみ。Google API リクエストの段階で fields パラメータを使い、件名・説明・参加者・場所・会議URL・添付などの「内容」はそもそも取得していません（プライバシー保護・全フェーズ共通の原則）。
